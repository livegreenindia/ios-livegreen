# Play Store Assets

This folder contains all graphic assets required for Google Play Store submission.

## Required Assets

### App Icon
- **hi-res-icon.png**: 512 x 512 px (32-bit PNG with alpha)
- Already generated by flutter_launcher_icons

### Feature Graphic
- **feature-graphic.png**: 1024 x 500 px
- Displayed at the top of your store listing
- Should represent your app's core value proposition
- No text is required as Google may overlay text

### Screenshots
Minimum 2, maximum 8 screenshots per device type.

#### Phone Screenshots (Required)
- **phone-1.png** to **phone-8.png**
- Dimensions: 16:9 aspect ratio recommended
- Minimum: 320px on shortest side
- Maximum: 3840px on longest side
- Common sizes: 1080 x 1920 px or 1242 x 2208 px

#### 7-inch Tablet Screenshots (Optional but recommended)
- **tablet-7-1.png** to **tablet-7-8.png**
- Dimensions: 1024 x 600 px minimum

#### 10-inch Tablet Screenshots (Optional but recommended)
- **tablet-10-1.png** to **tablet-10-8.png**
- Dimensions: 1280 x 800 px minimum

### Promo Video (Optional)
- YouTube video URL
- 30 seconds to 2 minutes recommended

## Screenshot Suggestions for LiveGreen

1. **Splash/Welcome Screen** - Show the branding
2. **Activity Page** - Show daily wellness activities
3. **Progress Charts** - Show the graphs and completion stats
4. **Community Forum** - Show social features

## Creating Screenshots

### Using Android Emulator
```bash
# Capture screenshot from emulator
adb exec-out screencap -p > screenshot.png
```

### Using Flutter
```bash
# Run on device/emulator and use:
# - Power + Volume Down on physical device
# - Camera icon in emulator sidebar
flutter run
```

### Using Screenshot Tools
- Recommended: Use a screenshot framing tool like:
  - https://screenshots.pro
  - https://mockuphone.com
  - https://previewed.app
